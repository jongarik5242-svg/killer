# killer
===================================================
           AI BLOCKER — ІНСТРУКЦІЯ З ВИКОРИСТАННЯ
===================================================

ЩО ЦЕ ТАКЕ
-----------
Скрипт стежить за заголовком активного вікна браузера.
Якщо виявляє AI-сайт (ChatGPT, Claude, Gemini тощо) —
миттєво закриває всі вікна браузера.
Працює тихо у фоні, стартує автоматично разом з Windows.


ФАЙЛИ
-----
  ai_blocker.py         — основний скрипт
  ai_blocker_start.vbs  — тихий запуск без вікна консолі
  install.bat           — інсталятор (запускати один раз)


ВСТАНОВЛЕННЯ
------------
1. Встанови psutil якщо ще не встановлено:
      pip install psutil

2. Поклади всі три файли в одну папку.

3. Клікни правою кнопкою на install.bat
   → "Запустити від імені адміністратора"

4. Перезапусти Windows.

Після цього блокер стартуватиме автоматично при кожному вході.


ВИМКНЕННЯ (тимчасово)
---------------------
Відкрий папку автозапуску:
  Win+R → shell:startup → Enter

Видали або перемісти звідти файл:
  ai_blocker_start.vbs

Перезапусти Windows — блокер більше не стартуватиме.


ВИМКНЕННЯ (повністю)
--------------------
1. Видали ai_blocker_start.vbs з папки автозапуску (див. вище)
2. Видали папку %USERPROFILE%\ai_blocker


ЛOGI
----
Всі спрацювання записуються у файл:
  %USERPROFILE%\ai_blocker\ai_blocker.log

Формат запису:
  2025-03-03 14:22:01: KILL — AI detected: ChatGPT - Google Chrome


ЗАБЛОКОВАНІ САЙТИ (за ключовими словами в заголовку)
----------------------------------------------------
  chatgpt / openai / gemini / bard /
  claude / deepseek / grok / copilot


ДОДАТИ НОВИЙ САЙТ ДО БЛОКУВАННЯ
--------------------------------
Відкрий ai_blocker.py у блокноті.
Знайди рядок:
  STOP_WORDS = [...]
Додай нове слово у список, наприклад:
  STOP_WORDS = ["chatgpt", "openai", ..., "perplexity"]


ВИМОГИ
------
  - Windows 10 / 11
  - Python 3.8+
  - psutil  (pip install psutil)
  - pywin32 (pip install pywin32)
